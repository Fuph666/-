# 蓝牙遥控小车 (Bluetooth RC Car)

基于 STM32F103C8T6 的多功能小车，支持蓝牙遥控、自动循迹、电量监测及低电量警报。

## 硬件清单

| 模块         | 型号                 | 数量     |
| ------------ | -------------------- | -------- |
| 主控         | STM32F103C8T6        | 1        |
| 电机驱动     | TB6612FNG            | 1        |
| 蓝牙模块     | HC-05                | 1        |
| 直流减速电机 | TT Motor             | 4        |
| 锂电池组     | 12.6V (3S)           | 1        |
| 降压模块     | 12V 转 5V 成品模块   | 1        |

| 蜂鸣器       | 有源蜂鸣器           | 1        |

| 超声波传感器 | HC-SR04              | 1 (预留) |

> 接线详见原理图，此处不再列出引脚分配。

```
                 └── 传感器
```

## 电池电量采集

采用 5.1K + 1K 电阻分压网络，将电池电压衰减至 STM32 ADC 量程内。

```
电池正极 → 5.1KΩ → ADC 引脚 → 1KΩ → GND
```

分压比：`V_ADC = V_bat × 1K / (5.1K + 1K) = V_bat / 6.1`

| 电池状态 | 电池电压 | ADC 采样电压 |
| -------- | -------- | ------------ |
| 满电     | 12.6V    | 2.07V        |
| 正常     | 11.1V    | 1.82V        |
| 电量低   | 10.5V    | 1.72V        |
| 过放临界 | 9.9V     | 1.62V        |

ADC 采样后通过公式反算实际电池电压：`V_bat = V_ADC × 6.1`

## 功能说明

### 蓝牙远程控制

通过手机蓝牙串口 APP（蓝牙调试器）发送单字符指令控制小车。

| 指令 | 动作                            |
| ---- | ------------------------------- |
| `F`  | 前进                            |
| `B`  | 后退                            |
| `L`  | 左转                            |
| `R`  | 右转                            |
| `S`  | 停止                            |
| `X`  | 模式切换（蓝牙控制 ↔ 自动循迹） |

> 控制方式为点按式：发送运动指令后小车执行动作，若 **200ms 内** 没有收到新指令，小车自动停止，防止失控。

上电默认进入**蓝牙控制模式**。在循迹模式下，仍可通过串口指令控制小车运动。

### 自动循迹

通过红外传感器（TCRT5000）识别黑色轨迹，自动纠正行驶方向，使小车沿预设轨迹稳定行驶。

发送 `X` 指令切换到循迹模式，再次发送 `X` 切回蓝牙控制模式。

### 自动避障 (未实现)

利用 HC-SR04 超声波传感器检测前方障碍物。当障碍物距离小于安全阈值时，自动执行转向或后退等避障动作。硬件已预留接口，待后续实现。

### 低电量警报

- 电池电压 **低于 10.5V** 时触发警报，蜂鸣器以 500ms 间隔间歇蜂鸣
- 电池电压 **低于 9.9V**（过放临界）时持续蜂鸣，并强制停车以保护电池
- 电量信息定时通过蓝牙回传至手机 APP

### TC6612 电机驱动

使用 TB6612FNG 双路电机驱动芯片，支持正反转及 PWM 调速，相比 L298N 具有更低的导通压降和更高的效率。

## 蓝牙调试器使用说明

手机端使用**蓝牙调试器** APP（浏览器搜索下载）。

### 连接步骤

1. 小车上电，HC-05 指示灯快闪表示等待连接
2. 打开蓝牙调试器 APP，点击加号进行搜索连接
3. 选择 HC-05 并配对（默认密码 1234）
4. 连接成功后指示灯变为慢闪

### 编辑按键指令

进入 APP 的编辑模式，按代码中的指令协议为按键绑定发送内容：

- 编辑 **UP** 按钮发送 `F`（前进）
- 编辑 **DOWN** 按钮发送 `B`（后退）
- 编辑 **LEFT** 按钮发送 `L`（左转）
- 编辑 **RIGHT** 按钮发送 `R`（右转）
- 新建按钮发送 `X`（模式切换）
- 新建按钮发送 `S`（停止）

### 蓝牙模块接线注意

蓝牙模块电源经过跳线连接，如果跳线脱落：**蓝色杜邦线接 VCC，棕色接 GND**。

## 开发环境

- **IDE**: STM32CubeIDE / Keil MDK v5
- **HAL 库**: STM32CubeF1 HAL Driver
- **代码生成**: STM32CubeMX
- **下载器**: ST-Link V2 (SWD)
- **串口调试**: SSCOM / 任意串口助手
- **手机 APP**: 蓝牙调试器

## 快速开始

1. 用 STM32CubeMX 打开项目配置，确认引脚分配与原理图一致
2. 编译并烧录到 STM32
3. 确认跳线连接正常（蓝 VCC / 棕 GND）
4. 上电，手机蓝牙调试器连接 HC-05
5. 编辑按键指令（F / B / L / R / S / X）后即可操控

## 注意事项

- HC-05 默认波特率 9600，若修改过请同步更新代码
- TB6612 的 VM（电机电源）和 VCC（逻辑电源）需分开供电
- ADC 采样建议做滑动平均滤波以消除电机干扰
- 12.6V 锂电池组过放会永久损坏，低电量警报触发后请尽快充电
- 循迹时确保环境光线稳定，黑色轨迹与背景对比度足够
